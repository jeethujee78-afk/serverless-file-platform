# Serverless File Platform 🚀

## 📌 Overview
This project is a serverless file upload system built using AWS services.

Users can upload files through a web interface, and the files are stored in Amazon S3 while metadata is stored in DynamoDB.

---

## 🏗️ Architecture
- Frontend (HTML + JavaScript)
- API Gateway (HTTP API)
- AWS Lambda (Python)
- Amazon S3 (File storage)
- DynamoDB (Metadata storage)

---

## 🔄 Workflow
1. User uploads file from browser
2. API Gateway receives request
3. Lambda function processes request
4. File stored in S3 bucket
5. Metadata saved in DynamoDB
6. Image/file URL generated and returned

---

## ⚙️ Technologies Used
- AWS Lambda
- Amazon S3
- DynamoDB
- API Gateway
- JavaScript (Frontend)
- Python (Backend)

---

## 🚀 Features
- Upload any file format
- Store metadata with timestamp
- Serverless architecture (no servers)
- Scalable and cost-efficient
